Water Recreation Checking Tool (REPORT)

The main application page includes:
- A leaflet map with markers at gauge locations
- A forecast graph that can be adjusted to display min, max, or average streamflow or stage
- A tool to check if the water is deep enough for various watercraft
- A tool to check for flooding

Instead of using the API for the reach forecast, I chose to use the gauge forecast. The benefit is that this enabled me to not only pull forecasts for streamflow, but also for water stage. Unfortunately it also disabled the ability to choose between short, medium, and long forecasts.

The flood estimations retrieved from an API key provided to the class appear to underpredict flood volumes at many location.
